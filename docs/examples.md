# NEURO Examples

For practical examples of how to use the NEURO language, please refer to:

1.  **The `examples/` directory:** This directory in the main repository contains various `.nr` script examples demonstrating different features.
2.  **`README.md` Quick Start:** The main README file includes a concise example to get you started.
3.  **`language_guide.md`:** This guide contains syntax examples for specific language features.
